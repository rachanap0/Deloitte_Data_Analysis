This repository contains my work and reflections from completing Deloitte Australia’s Data Analytics Job Simulation on the Forage platform. The simulation provided a hands-on experience of the type of work performed by Deloitte’s analytics and forensic technology teams.
**Overview**
During this simulation, I:
- Completed a Deloitte job simulation involving data analysis and forensic technology.
- Created a data dashboard using Tableau to visualize insights effectively.
- Used Excel to classify data and draw actionable business conclusions.
This experience helped me understand Deloitte’s approach to solving complex business problems using technology and analytics.
**Skills Practiced**
Dashboard Creation and Data Visualization (Tableau)
Data Classification and Business Decision Making (Excel)
Understanding real-world consulting analytics workflows
**Why I Did This**
I participated in this simulation to gain a realistic understanding of Deloitte’s work environment and to strengthen my practical data analytics skills. It confirmed my interest in working on technology-driven solutions for clients and motivated me to apply these skills in a professional setting.

**Takeaways**
Real-world consulting projects require both technical and analytical skills to derive insights from data.
Tools like Tableau and Excel are crucial for visualizing and interpreting data effectively.
Practical simulations help bridge the gap between theory and professional practice.
